# ATAC-seq 


* 使用Tn5转酶切割染色质的开放区域、对这些片段进行测序。    
* 优点：只需要少量样本(500~50000 cell)、
* 可结合基因表达数据、挖掘开放区域的潜在转录因子结合区域(TFBS)，得到关键的TF
    - Atlas绘制
    - 潜在的疾病Marker

## Pipeline

* Sequencing QC & Trim
* Mapping
* Pre-analysis
    - Mark PCR Duplicate
    - Filter Insert Size
* Peak Calling
    - QC & replicate reproducibility
    - High Quality Peak Sets
* Analysis
    - Normalization
    - Peak Annotation
    - Motif detection
    - GO, Pathway ...
